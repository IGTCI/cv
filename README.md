# 🌐 Personal CV Website

A responsive personal portfolio website built with HTML and CSS, showcasing my profile, skills, and contact information.

## 📸 Preview

The site includes:
- **Hero Section** — Animated cloud and mountain visuals with a welcoming introduction
- **Profile Section** — Avatar, bio, and a brief introduction
- **Skills Section** — Design & Development skills with visual elements
- **Contact Section** — Call-to-action button to get in touch
- **Footer** — Social media links (LinkedIn, Twitter)

## 🛠️ Technologies Used

| Technology | Usage |
|---|---|
| **HTML5** | Page structure and semantic markup |
| **CSS3** | Styling, layout, gradients, and hover effects |
| **Google Fonts** | Merriweather, Montserrat, Sacramento |

## 📁 Project Structure

```
cv/
├── index.html          # Main page
├── CSS/

│   └── gtc-logo.png    # Site logo
└── Images/
    ├── GTC-avatar.png  # Profile picture
    ├── cloud1.png      # Cloud graphic
    ├── mountain3.png   # Mountain graphic
    ├── coding.png      # Coding illustration
    └── siting.png      # Work illustration
```

## 🚀 How to Run

Simply open `index.html` in your web browser — no server or build step required.

```bash
# Clone and open
git clone https://github.com/IGTCI/cv.git
cd cv
start index.html
```
